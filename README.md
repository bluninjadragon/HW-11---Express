# HW 11 Note Taker

## Description

To create the backend routes for an Note Taking app to render the proper updated pages and notes

## Installation

On the command line, run "npm i" and then "npm start"

## Usage

Users will be able to create and add a note. This will then be saved on the left hand bar for retrieval. This is done using the get and post routes created on the server side using Express.
